# COTOBA Agent sample application for Android

## Overview
This is an Android sample application for COTOBA Agent.  
This application uses Android speech recognition and speech synthesis to perform interactive processing.  
The result of the interaction with the bot is displayed in the chat app style.  

## About setting up the endpoint, userid and api-key
You need to set the endpoint URL and api key for the bot.  
Specifies the endpoint url and api key for the bot that accesses the application/src/main/re/value strings.xml.  
Also, if you want to use a specific user ID, change the userid in strings.xml.
